# Portfolio

A beautiful scrolling portfolio website built with React, featuring modern gradients, smooth scrolling, and animations.

## Features

- Smooth scrolling between sections
- Gradient backgrounds (red to blue theme)
- Fade-in animations on scroll using Framer Motion
- Responsive design for PC and mobile
- Sections: Hero, About, Projects, Skills, Contact

## Getting Started

1. **Install Node.js**: Download and install Node.js LTS from [https://nodejs.org/](https://nodejs.org/)

2. **Navigate to the project folder**:
   ```
   cd "c:\Users\Administrator\Downloads\AI-T-NGHI-N-C-U-AI-AI-AUTO-RESEARCH-\portfolio"
   ```

3. **Install dependencies**:
   ```
   npm install
   ```

4. **Start the development server**:
   ```
   npm start
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser to view the portfolio.

## Customization

- Edit the content in `src/components/` files to personalize the portfolio.
- Modify styles in `src/App.css` to change colors, fonts, or layouts.
- Add images to `public/` folder and reference them in components.

## Build for Production

Run `npm run build` to create a production build in the `build` folder.
